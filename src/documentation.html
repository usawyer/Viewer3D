Documentation of 3DViewer_v2.0

Introduction

This is the project that implement a program to view 3D wireframe models (3D Viewer) in the C programming language.

The models themselves must be loaded from .obj files and be viewable on the screen with the ability to rotate, scale and translate.

The program allows to change settings and save the captured images and create gif-animations.

Installation

Can be opened in QT creator or create the executable using make install.

Description

The app supports the following features.

  • load a wireframe model from an obj-file
  • translate the model in relation to the X, Y, Z axes
  • rotate the model relative to its X, Y, Z axes
  • scale the model by a given value to its X, Y, Z axes
  • output model information: file name, number of vertices and edges
  • customize model
  • save settings between program restarts
  • save images
  • make gif-animations

How to use

  1. To load the model, click "open file" and select the required file.
  2. To translate, rotate or scale the model, you can either click on the required button "<" / ">" or hold down the buttons "<" / ">" for the corresponding axis or set the desired value in the box. If you change the value in the box then before doing other transformations reset to zero the value in the spinbox.
  3. To change the projection type in the "type of projection" field, select the desired projection type: central or parallel.
  4. The "settings of the edges" field allows to:
    • change the type of drawn line (solid or dashed)
    • change the color of the lines by clicking on the "choose..." button and selecting the appropriate color, then clicking "OK"
    • change the line thickness (1.0, 2.5 or 5.0)
  5. The "settings of the vertices" field allows to:
    • change the display method (none, circle, square)
    • change the color of the points by clicking on the "choose..." button and selecting the appropriate color, then clicking "OK"
    • change the size of the points (5.0, 10.0 or 15.0).
  6. The "set background color" field allows you to change the color of the background by clicking on the "choose..." button and selecting the appropriate color and then clicking "OK".
  7. To take a screenshot:
    • click the appropriate button
    • write a name, and select the location that file is being saved to and choose the required format: .bmp or .jpeg
    • press "Save".
  8. To make a GIF-animation:
    • press the appropriate button
    • change the model
    • write a name, and select the location that file is being saved to
    • press "Save".

Notes

  1. Don't use large files. Up to 10 Mb.
  2. Don't enter incorrect types to save the file.
